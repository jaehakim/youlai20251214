## 프로젝트 소개

[vue3-element-admin](https://gitcode.com/youlai/vue3-element-admin) Vue3, Vite7, TypeScript 및 Element-Plus를 기반으로 구축된 초간단 즉시 사용 가능한 엔터프라이즈급 백오피스 관리 프론트엔드 템플릿. 매칭 Java 백엔드 [youlai-boot](https://gitee.com/youlaiorg/youlai-boot) 및 Node 백엔드 [youlai-nest](https://gitee.com/youlaiorg/youlai-nest) 。 개발 간소화 버전[vue3-element-template](https://gitee.com/youlaiorg/vue3-element-template) 및 JS 버전[vue3-element-admin-js](https://gitee.com/youlaiorg/vue3-element-admin) 供开发者快速开发。


## 프로젝트 특징

- **간결하고 사용하기 쉬움**：기반 [vue-element-admin](https://gitee.com/panjiachen/vue-element-admin) 升级的 Vue3 版本，无过渡封装 ，易上手。
- **데이터 상호작용**： 지원 `Mock` 데이터 및[온라인 인터페이스 문서](https://www.apifox.cn/apidoc/shared-195e783f-4d85-4235-a038-eec696de4ea5)，并提供配套的 [Java](https://gitee.com/youlaiorg/youlai-boot) 和 [Node](https://gitee.com/youlaiorg/youlai-nest) 백엔드 소스코드 제공.

- **系统功能：** 사용자 관리, 역할 관리, 메뉴 관리, 부서 관리, 사전 관리, 시스템 설정, 공지사항 등 기능 모듈 제공.
- **权限管理：** 지원动态路由、按钮权限、角色权限和数据权限等多种权限管理方式。

- **基础设施：** 提供国际化、多布局、暗黑模式、全屏、水印、인터페이스 문서和代码生成器等功能。
- **지속적인 업데이트**：프로젝트持续开源更新，实时更新工具和依赖。

## 프로젝트启动

- **환경 준비**

| 环境类型 | 버전 요구사항 | 비고 |
|---------|---------|------|
| **Node.js** | `^20.19.0` 或 `>=22.12.0` | 推荐使用 LTS 版本（主版本을偶数） |
| **패키지 매니저** | `pnpm >= 8.0.0` | 프로젝트使用 pnpm 作을패키지 매니저 |
| **开发工具** | [Visual Studio Code](https://code.visualstudio.com/Download) | 推荐安装 Vue、TypeScript 相关插件 |


- **빠른 시작**

```bash
# 코드 복제
git clone https://gitee.com/youlaiorg/vue3-element-admin.git

# 切换目录
cd vue3-element-admin

# pnpm 설치
npm install pnpm -g

# 미러 소스 설정(선택 사항)
pnpm config set registry https://registry.npmmirror.com

# 종속성 설치
pnpm install

# 启动运行
pnpm run dev
```


## 프로젝트部署

실행 `pnpm run build` 命令后，프로젝트将被打包并生成 `dist` 目录。接下来，将 `dist` 디렉토리의 파일을 서버 `/usr/share/nginx/html` 目录下，并配置 Nginx 进行反向代理。

```bash
pnpm run build
```

以下是 Nginx 的配置示例：

```nginx
server {
    listen      80;
    server_name localhost;

    location / {
        root   /usr/share/nginx/html;
        index  index.html index.htm;
    }

    # 反向代理配置
    location /prod-api/ {
        # 请将 api.youlai.tech 替换을您的后端 API 주소，并注意保留后面的斜杠 /
        proxy_pass http://api.youlai.tech/;
    }
}
```

更多详细信息，请参考这篇文章：[Nginx 설치 및 구성](https://blog.csdn.net/u013737132/article/details/145667694)。

## 로컬 Mock

프로젝트同时지원在线和本地 Mock 接口，默认使用线上接口，如需替换을 Mock 接口，수정文件 `.env.development` 的 `VITE_MOCK_DEV_SERVER` 을  `true` **即可**。

## 백엔드 인터페이스

> 如果您具备Java开发基础，按照以下步骤将在线接口转을本地백엔드 인터페이스，创建企业级前后端分离开发环境，助您走向全栈之路。

1. 获取기반 `Java` 和 `SpringBoot` 开发的后端 [youlai-boot](https://gitee.com/youlaiorg/youlai-boot.git) 소스코드를 가져옵니다.
2. 根据后端工程的说明文档 [README.md](https://gitee.com/youlaiorg/youlai-boot#%E9%A1%B9%E7%9B%AE%E8%BF%90%E8%A1%8C) 完成本地启动。
3. 수정 `.env.development` 파일의 `VITE_APP_API_URL` 的值，将其从 https://api.youlai.tech 更改을 http://localhost:8989 即可。


## 注意事项

- **自动导入插件自动生成默认关闭**

  模板프로젝트的组件类型声明已自动生成。如果添加和使用新的组件，请按照图示方法开启自动生成。在自动生成完成后，记得将其设置을 `false`，避免重复실행引发冲突。

  ![](https://foruda.gitee.com/images/1687755823137387608/412ea803_716974.png)

- **프로젝트启动浏览器访问空白**

  请升级浏览器尝试，低版本浏览器内核可能不지원某些新的 JavaScript 语法，比如可选链操作符 `?.`。

- **프로젝트同步仓库更新升级**

  프로젝트同步仓库更新升级之后，建议 `pnpm install` 安装更新依赖之后启动 。

- **프로젝트组件、函数和引用爆红**

	VSCode 재시작 시도

- **其他问题**

  如果有其他问题或者建议，建议 [ISSUE](https://gitee.com/youlaiorg/vue3-element-admin/issues/new)
